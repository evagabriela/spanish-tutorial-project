Una página web está compuesta por dos partes: el front-end y el back-end. El front-end se refiere a la parte visible de la página web. Es decir, la información, fotos, imágenes, links, formularios, y otros elementos visibles con los cuales uno interactúa al abrir una página web. Por otro lado, el back-end no es visible para el usuario. A grandes rasgos, el back-end es la parte que interactúa con servidores y base de datos que envían información al front-end cuando este lo necesita.

En este curso nos enfocaremos en el front-end. Es decir, aprenderemos sobre las tres tecnologías que se necesitan para construirlo: HTML, CSS y JavaScript. Para que tengas una mejor idea, imagina que vas a construir una casa. Para empezar, necesitarás construir la estructura de la casa, osea, los cimientos, paredes, techo. Luego tendrás que agregarle más estilo a la casa. Puedes pintar las paredes de amarillo, blanco, o gris. También tienes que elegir los colores del piso, los cuadros, el tamaño de las mesas, y todos los demás detalles que hacen que tu casa tenga un estilo único. Por último, deberás atender los detalles más funcionales. Por ejemplo, cómo va a funcionar el portón eléctrico, la ducha de la bañera, o el encendido de las luces del jardín.

Para la construcción del front-end de una página web necesitamos enfocarnos en los mismos procedimientos que el ejemplo de la casa. Primero, la estructura o el esqueleto del sitio web. Para eso se utiliza HTML. Luego, para darle estilo al sitio se utiliza CSS, con el cual puedes poner colores, bordes o cambiar el tamaño de las letras, entre otras cosas. Finalmente, JavaScript te da la posibilidad de agregar interacciones a la página como por ejemplo, que aparezca una nueva ventana cada vez que el usuario hace click en algún botón.
