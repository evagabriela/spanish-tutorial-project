Una página web está compuesta por dos partes: el front-end y el back-end. El front-end se refiere a la parte visible de la página web. Es decir, la información, fotos, imágenes, links, formularios, y otros elementos visibles con los cuales uno interactúa al abrir una página web como Facebook o Twitter. Por otro lado, el back-end no es visible para el usuario. A grandes rasgos, el back-end es la parte que interactúa con servidores y base de datos que envían información al front-end cuando este lo necesita.

En este curso nos enfocaremos en el front-end. Es decir, aprenderemos sobre las tres tecnologías que se necesitan para construirlo: HTML, CSS y JavaScript. Para que tengas una mejor idea, imagina que vas a construir una casa. Para empezar, necesitarás construir la estructura de la casa, osea, los cimientos, paredes, techo. Luego tendrás que agregarle más estilo a la casa. Puedes pintar las paredes de amarillo, blanco, o gris. También tienes que elegir los colores del piso, los cuadros, el tamaño de las mesas, y todos los demás detalles que hacen que tu casa tenga un estilo único. Por último, deberás atender los detalles más funcionales. Por ejemplo, cómo va a funcionar el portón eléctrico, la ducha de la bañera, o el encendido de las luces del jardín.

Para la construcción del front-end de una página web necesitamos enfocarnos en los mismos procedimientos que el ejemplo de la casa. Primero, la estructura o el esqueleto del sitio web. Para eso se utiliza HTML. Luego, para darle estilo al sitio se utiliza CSS, con el cual puedes poner colores, bordes o cambiar el tamaño de las letras, entre otras cosas. Finalmente, JavaScript te da la posibilidad de agregar interacciones a la página como por ejemplo, que aparezca una nueva ventana cada vez que el usuario hace click en algún botón.

Empecemos por la estructura del sitio web. HTML trabaja con etiquetas. Generalmente, las etiquetas vienen de a dos: una etiqueta de inicio < > y una etiqueta de cierre </>. Entre las etiquetas se coloca el texto u otro contenido.
Ejemplo: <h1> Hola! Estoy dentro de etiquetas </h1>

Cómo crear títulos?

Para empezar a construir un sitio web necesitamos un título, unos párrafos, y también podemos agregar una imágen y unos links. HTML te permite escribir títulos con los siguientes elementos: h1, h2, h3, h4, h5, h6. La "h" se refiere a "heading" en inglés, osea título en español. Los números que acompañan la "h" indican el tamaño del título, siendo h1, el título principal o el más grande, seguido por h2, que sería el subtítulo, y luego los títulos más pequeños que se utilizan para dividir secciones.

<!DOCTYPE html>
<html>
  <body>
    
    <h1>Soy el título principal</h1>
    <h2>Soy un subtítulo</h2>
    
  </body>
</html>

Cómo crear párrafos?

Para escribir párrafos se utilizan estas etiquetas <p> </p> con el contenido del párrafo adentro. De más está decir que la "p" corresponde a la inicial de la palabra párrafo.

<!DOCTYPE html>
<html>
  <body>
    
    <p>Soy un párrafo!</p>
    <p>Puedes agregar cuantos párrafos quieras!</p>
    
  </body>
</html>

En el editor de abajo podrás ver la diferencia de tamaños entre cada título. Por ahora, sólo enfocate en las etiquetas de títulos que están dentro de las etiquetas del <body>. Juega un poco y explora con los demás títulos. Escribe otros títulos debajo de los ya existentes utilizando h4, h5 y h6.

Luego escribe uno o dos párrafos en el editor. Puedes escribir tus párrafos debajo de los títulos. No te olvides de utilizar las etiquetas correspondientes.

Recuerda hacer click en "Output" para examinar cómo se ve el resultado en la web.

Para acompañar al contenido escrito de la página, podemos usar distintas imágenes. Para agregar una imágen utilizamos el elemento img, pero a diferencia de los títulos y párrafos, img no requiere una etiqueta de cierre </>.

El elemento img no funciona por si solo; necesita apuntar a un URL específico o una dirección determina donde se encuentra la imágen que se quiere utilizar. Esto se hace a través del atributo src que es la versión corta del inglés "source" que significa fuente, es decir el lugar donde se encontrará la imágen.

Ejemplo: <img src = “https://www.tuimagen.com/tu-imagen.jpg">.

Sin embargo, todavía falta algo. Todas las etiquetas de imágenes necesitan incorporar un atributo llamado alt. Este atributo contiene el texto que se mostrará en el caso de que la imágen no se cargue debidamente y no pueda mostrarse en la página. Además, alt sirve para hacer más accesible la página web.

La forma correcta de colocar una imágen es la siguiente:
<img src = “https://www.tuimagen.com/tu-imagen.jpg" alt=“Descripción de la imágen”>

Algunas imágenes necesitarán unos retoques en el tamaño. Para especificar el tamaño de la imágen se utiliza width que significa "ancho" en español.

<!DOCTYPE html>
<html>
  <body>
    
    <img src="https://www.tuimagen.com/tu-imagen.jpg" alt="Descripción de la imágen" width=200px>
    
  </body>
</html>

En el ejemplo del editor encontramos la imágen de Harry Potter que tiene un ancho de 300 pixeles (fijate en la etiqueta de imágen, después del atributo alt). Haz click en "Output" para ver el resultado.

Prueba agregar una imágen más en el editor. Andá a Google y buscá una imágen que te guste. Si la imágen es muy grande, puedes modificarla agregando "width" y el número de pixeles que prefieras.

Recuerda: una vez que hayas encontrado la imágen, tienes que hacer click en el botón "Ver imágen" y copiar ese link para ponerlo dentro de la etiqueta img.

Los enlaces o links se utilizan para “enlazar” ciertas páginas al sitio web. Por ejemplo, si el contenido de una página web es acerca del cambio climático, y quieres que los usuarios de la página visiten otras páginas para conocer más acerca del cambio climático, entonces necesitarás links. Los links están representados con la etiqueta <a> </a>. La "a" es la inicial de la palabra en inglés "anchor".

Al igual que las imágenes, los links también necesitan un atributo que indique dónde se encuentra el enlace. Este atributo se llama href, versión corta del inglés "Hypertext reference" que significa hypertexto de referencia. Otro atributo importante es target, al cual se le agrega el valor "_blank". Gracias a este atributo, cada vez que el usuario hace click en el link, la nueva página se abre en una nueva pestaña.

Finalmente, entre la etiqueta de inicio y la etiqueta de cierre se coloca el texto sobre el cual se deberá hacer click para acceder al link. Un link se representa de la siguiente manera:

<!DOCTYPE html>
<html>
  <body>
    
    <a href= “https://www.google.com/” target="_blank">Haz click aquí! Soy un link apuntando a Google</a>
    
  </body>
</html>

El editor contiene un link de información acerca del cambio climático cuyo resultado podrás ver en la pantalla de "Output". Haz click en el link y luego crea tu propio link. No te olvides de agregar el atributo target="_blank" antes del final de la etiqueta de apertura

HTML nos permite crear dos tipos de listas: ordenadas y no ordenadas. Las listas ordenadas están numeradas. Se representan con la etiqueta <ol> </ol> versión corta del inglés "ordered list". Dentro de estas etiquetas se colocan otras etiquetas para enumerar la lista, estas son <li> </li>. Las listas desordenadas representadas por <ul> </ul>, donde ul es la versión corta del inglés "unordered list", crean listas que empiezan con puntos.

<!DOCTYPE html>
<html>
  <body>
    
    <ol>
      <li>Soy una lista ordenada<li>
      <li>Estoy numerada<li>
      <li>Aquí importan las prioridades<li>
    </ol>

    <ul>
      <li>Soy una lista desordenada<li>
      <li>No estoy numerada<li>
      <li>Tengo puntos<li>
    </ul>
    
  </body>
</html>

En el editor se encuentran los dos tipos de listas. Agregá dos puntos más a cada una de las listas. Luego, haz click en "Output" para ver el resultado.

Ya tenemos una idea de cómo estructurar una página web, con títulos, párrafos, imágenes, links y listas. Ahora es tiempo de agregarle un poco de estilo a la página, es decir, es tiempo de CSS.

Para empezar podemos cambiar el color del titulo principal. Para eso, se debe seleccionar el elemento que queremos cambiar, en este caso h1. Para cambiar estilos, CSS no requiere utilizar estos signos < > < / > . Sin embargo, CSS si requiere otros símbolos como las llaves { }.

En el siguiente ejemplo, seleccionamos el elemento que queríamos cambiar, h1, y luego le agregamos el estilo, es decir el color, y le asignamos el color rojo (red). Nunca te olvides de agregar un punto y coma al final de cada estilo. Los estilos van siempre dentro de llaves.


h1 {
  color: red;
}

También podemos modificar el color del fondo osea el "background" en inglés. Para eso, seleccionamos un elemento, por ejemplo la p, y le cambiamos el estilo dentro de las llaves. En el ejemplo de abajo cambiamos el color del background a azul (blue).


p {
  background-color: blue;
}

Observación: Hasta ahora hemos trabajado con un solo panel: HTML. Sin embargo, los estilos se agregan en otro archivo, diferente al archivo donde se encuentra el contenido HTML. Es por eso, que nuestro editor nos muestra un panel diferente para CSS. En el caso de que no veas este panel, puedes hacer click en la pestaña que dice CSS.

Cambia el color de la h1 por el color "blue" en el panel de CSS. También modifica el color del background del párrafo con el color "yellow". Haz click en "Output" para ver los cambios.

JS Bin on jsbin.com

Podemos cambiar varios estilos a la vez, colocando todos estos estilos dentro de la misma llave. Por ejemplo, para cambiar el tamaño y tipo de letra del texto, podemos utilizar los estilos font-size y font-family.


p {
  font-size: 20px;
  font-family: Monospace;
}

¡El cielo es el límite!

También podemos agregar bordes a nuestros elementos en HTML. Por ejemplo, si queremos agregar un borde a la imágen, entonces seleccionamos img y agregamos los estilos que queramos dentro de las llaves { }. Los bordes tienen tres atributos: color, ancho y estilo.


img {
  border-color: red;
  border-width: 2px;
  border-style: solid;
}
