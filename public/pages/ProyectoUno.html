Objetivos

En este curso te guiaremos paso a paso a través del proceso de creación de una página web.
Los objetivos de este curso son los siguientes:

1. Al finalizar el curso serás capaz de construir una página web sencilla

2. Podrás explicar la diferencia entre HTML y CSS

3. Serás capaz de utilizar iconos y diferentes estilos de fuente a través de Google Fonts

4. Podrás construir la estructura y estilos del sitio web de una manera ordenada respetando las mejores prácticas

Aquí podras descargar el proyecto completo en el caso de que quieras compararlo con tu trabajo. Sin embargo, te recomendamos que mires los videos y trates de hacer el proyecto paso a paso, sin copiar el resultado final. Descargar proyecto final
