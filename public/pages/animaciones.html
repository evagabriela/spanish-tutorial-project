Las pseudo clases nos permiten crear un estado predefinido de un elemento al que se le puede aplicar un estilo independiente. En otras palabras, podemos utilizar CSS para poder aplicar una característica especifica a un elemento.

Muchos elementos en HTML disponen de estados especiales o usos asociados a ellos, lo cual nos permite aplicar estilos diferentes a nuestros elementos en el HTML.

Por ejemplo, el elemento <a> </a> tiene cuatro estados:

  • Normal: Su estado normal
  • Visited: Cuando ya hemos visitado el link al que hace referencia
  • Hover: Cuando tenemos el cursor situado encima del mismo
  • Active: Cuando hacemos click sobre él

La funcionalidad de las pseudo clases ya se encuentra ampliamente implementada en la mayoría de los navegadores actuales. Esto nos permite aplicar una característica específica a un elemento a través de una clase definida en el CSS. Existen cuatro tipos diferentes de pseudo clases:

  • Links: Estas pseudo clases se usan para dar estilo al enlace tanto en su estado normal por defecto como cuando ya ha sido visitado, mientras mantenemos el cursor encima de él o cuando hacemos click en él
  • Dinámicas: Estas pseudo clases pueden ser aplicadas a cualquier elemento para definir como se muestran cuando el cursor está situado sobre ellos, o haciendo click en ellos o bien cuando son seleccionados. Por ejemplo :active
  • Estructurales: Permiten dar estilo a elementos basándonos en una posición numérica exacta del elemento. Por ejemplo :first-child, :nth-child
  • Otros:Algunos elementos pueden ser estilizados de manera diferente basándonos en el lenguaje o que tipo de etiqueta no son

Además, las pseudo-clases siempre están precedidas por dos puntos (:). Luego viene el nombre de la pseudo-clase y a veces un valor entre paréntesis. Ejemplo: :nth-child.

Dejemos a un lado la teoria y veamos como son las pseudo-clases:


                    a:link {
                      color: red;
                    }

                    a:visited {
                      color: blue;
                    }

                    a:hover {
                      color: gray;
                    }

                    li:first-child {
                      color: green;
                    }

Utilizar transiciones nos brinda la opción de poder agregar animaciones simples y elegantes a nuestro sitio web. Al mismo tiempo también nos permite tener una manera de controlar el tiempo de la animación cuando las propiedades cambian.

Anteriormente hemos hablado de pseudo clases, y por ejemplo usamos el estado :hover , la cual es una manera simple de aplicar transiciones con las pseudo clases de CSS. En el siguiente ejemplo vamos a utilizar :hover y también vamos a especificar el nombre de la propiedad, el tiempo de transición y una de las funciones de tiempo de transición el cual usamos linear

See the Pen RgzBqZ by eku (@eku) on CodePen.

En el ejemplo que vimos, cuando la pseudo clase :hover es activada, la altura del elemento cambia linearmente de 100px a 200px en 2 segundos.

Por lo tanto vemos que podemos hacer que el cambio de una propiedad pueda darse en un tiempo especifico. En la siguiente linea se puede ver la estructura básica del formato de como utilizar transiciones:

                  
  .ejemplo {
  transition: [propiedad] [tiempo] [tipo de función] [tiempo de retraso]
  }
                  
                
                  
  .button {
  transition: all .5s ease-in-out;
  }
