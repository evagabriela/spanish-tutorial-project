Empecemos por la estructura del sitio web. HTML trabaja con etiquetas. Generalmente, las etiquetas vienen de a dos: una etiqueta de inicio < > y una etiqueta de cierre </>. Entre las etiquetas se coloca el texto u otro contenido.
Ejemplo: <h1> Hola! Estoy dentro de etiquetas </h1>

Cómo crear títulos?

Para empezar a construir un sitio web necesitamos un título, unos párrafos, y también podemos agregar una imágen y unos links. HTML te permite escribir títulos con los siguientes elementos: h1, h2, h3, h4, h5, h6. La "h" se refiere a "heading" en inglés, osea título en español. Los números que acompañan la "h" indican el tamaño del título, siendo h1, el título principal o el más grande, seguido por h2, que sería el subtítulo, y luego los títulos más pequeños que se utilizan para dividir secciones.

Cómo crear párrafos?

Para escribir párrafos se utilizan estas etiquetas <p> </p> con el contenido del párrafo adentro. De más está decir que la "p" corresponde a la inicial de la palabra párrafo.

Para acompañar al contenido escrito de la página, podemos usar distintas imágenes. Para agregar una imágen utilizamos el elemento img, pero a diferencia de los títulos y párrafos, img no requiere una etiqueta de cierre </>.

El elemento img no funciona por si solo; necesita apuntar a un URL específico o una dirección determina donde se encuentra la imágen que se quiere utilizar. Esto se hace a través del atributo src que es la versión corta del inglés "source" que significa fuente, es decir el lugar donde se encontrará la imágen.

Ejemplo: <img src = “https://www.tuimagen.com/tu-imagen.jpg">.

Sin embargo, todavía falta algo. Todas las etiquetas de imágenes necesitan incorporar un atributo llamado alt. Este atributo contiene el texto que se mostrará en el caso de que la imágen no se cargue debidamente y no pueda mostrarse en la página. Además, alt sirve para hacer más accesible la página web.

La forma correcta de colocar una imágen es la siguiente:
<img src = “https://www.tuimagen.com/tu-imagen.jpg" alt=“Descripción de la imágen”>

Algunas imágenes necesitarán unos retoques en el tamaño. Para especificar el tamaño de la imágen se utiliza width que significa "ancho" en español.

Los enlaces o links se utilizan para “enlazar” ciertas páginas al sitio web. Por ejemplo, si el contenido de una página web es acerca del cambio climático, y quieres que los usuarios de la página visiten otras páginas para conocer más acerca del cambio climático, entonces necesitarás links. Los links están representados con la etiqueta <a> </a>. La "a" es la inicial de la palabra en inglés "anchor".

Al igual que las imágenes, los links también necesitan un atributo que indique dónde se encuentra el enlace. Este atributo se llama href, versión corta del inglés "Hypertext reference" que significa hypertexto de referencia. Otro atributo importante es target, al cual se le agrega el valor "_blank". Gracias a este atributo, cada vez que el usuario hace click en el link, la nueva página se abre en una nueva pestaña.

HTML nos permite crear dos tipos de listas: ordenadas y no ordenadas. Las listas ordenadas están numeradas. Se representan con la etiqueta <ol> </ol> versión corta del inglés "ordered list". Dentro de estas etiquetas se colocan otras etiquetas para enumerar la lista, estas son <li> </li>. Las listas desordenadas representadas por <ul> </ul>, donde ul es la versión corta del inglés "unordered list", crean listas que empiezan con puntos.
