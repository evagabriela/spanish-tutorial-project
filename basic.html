Una página web está compuesta por dos partes: el front-end y el back-end. El front-end se refiere a la parte visible de la página web. Es decir, la información, fotos, imágenes, links, formularios, y otros elementos visibles con los cuales uno interactúa al abrir una página web como Facebook o Twitter. Por otro lado, el back-end no es visible para el usuario. A grandes rasgos, el back-end es la parte que interactúa con servidores y base de datos que envían información al front-end cuando este lo necesita.

En este curso nos enfocaremos en el front-end. Es decir, aprenderemos sobre las tres tecnologías que se necesitan para construirlo: HTML, CSS y JavaScript. Para que tengas una mejor idea, imagina que vas a construir una casa. Para empezar, necesitarás construir la estructura de la casa, osea, los cimientos, paredes, techo. Luego tendrás que agregarle más estilo a la casa. Puedes pintar las paredes de amarillo, blanco, o gris. También tienes que elegir los colores del piso, los cuadros, el tamaño de las mesas, y todos los demás detalles que hacen que tu casa tenga un estilo único. Por último, deberás atender los detalles más funcionales. Por ejemplo, cómo va a funcionar el portón eléctrico, la ducha de la bañera, o el encendido de las luces del jardín.

Para la construcción del front-end de una página web necesitamos enfocarnos en los mismos procedimientos que el ejemplo de la casa. Primero, la estructura o el esqueleto del sitio web. Para eso se utiliza HTML. Luego, para darle estilo al sitio se utiliza CSS, con el cual puedes poner colores, bordes o cambiar el tamaño de las letras, entre otras cosas. Finalmente, JavaScript te da la posibilidad de agregar interacciones a la página como por ejemplo, que aparezca una nueva ventana cada vez que el usuario hace click en algún botón.

Empecemos por la estructura del sitio web. HTML trabaja con etiquetas. Generalmente, las etiquetas vienen de a dos: una etiqueta de inicio < > y una etiqueta de cierre </>. Entre las etiquetas se coloca el texto u otro contenido. Ejemplo: <h1> Hola! Estoy dentro de etiquetas </h1>

Cómo crear títulos?

Para empezar a construir un sitio web necesitamos un título, unos párrafos, y también podemos agregar una imágen y unos links. HTML te permite escribir títulos con los siguientes elementos: h1, h2, h3, h4, h5, h6. La h se refiere a heading en inglés, osea título. Los números que acompañan la h indican el tamaño del título, siendo h1, el título principal o el más grande, seguido por h2, que sería el subtítulo, y luego los títulos más pequeños que se utilizan para dividir secciones.

Ejemplos:

<h1> Soy el título principal </h1>

<h2> Yo soy un subtítulo </h2>

<h3> Esta es una sección </h3>

Cómo crear párrafos?

Luego de los títulos, necesitaremos unos párrafos. Para eso se utilizan estas etiquetas <p> </p> con el contenido del párrafo adentro. De más está decir que la "p" es la inicial de la palabra párrafo.

Ejemplo:

<p>Hola, soy un párrafo. Nunca te olvides de que generalmente al final del contenido siempre tiene que haber una etiqueta de cierre.</p>

Para acompañar al contenido escrito de la página, podemos usar distintas imágenes. Para agregar una imágen utilizamos el elemento img, pero a diferencia de los títulos y párrafos, img no requiere una etiqueta de cierre. El elemento img necesita apuntar a un URL específico o una dirección determina donde se encuentra la imágen que se quiere utilizar. Esto se hace a través del atributo scr que es la versión corta del inglés source que significa fuente, es decir el lugar donde se encontrará la imágen.

<img src = “https://www.tuimagen.com/tu-imagen.jpg">

Sin embargo, todavía falta algo. Todas las etiquetas de imágenes necesitan incorporar un atributo llamado alt. Este atributo contiene el texto que se mostrará en el caso de que la imágen no se cargue debidamente y no pueda mostrarse en la página. Además, alt sirve para hacer más accesible la página web. La forma correcta de colocar una imágen es la siguiente:

<img src = “https://www.tuimagen.com/tu-imagen.jpg" alt=“Descripción de la imágen”>

Los enlaces o links se utilizan para “enlazar” ciertas páginas al sitio web. Por ejemplo, si el contenido de una página web es acerca del cambio climático, y quieres que los usuarios de la página visiten otras páginas para conocer más acerca del cambio climático, entonces necesitarás links. Los links están representados con la etiqueta <a>. La "a" es la inicial de la palabra en inglés "anchor".

Al igual que las imágenes, los links también necesitan un atributo que indique dónde se encuentra el enlace. Este atributo se llama href, versión corta del inglés "Hypertext reference" que significa hypertexto de referencia. Un link se representa de la siguiente manera:

<a href= “https://www.google.com/”>Haz click aquí! Soy un link apuntando a la página de Google</a>

Entre la etiqueta de inicio y la etiqueta de cierre se coloca el texto sobre el cual se deberá hacer click para acceder al link.

HTML nos permite crear dos tipos de listas: ordenadas y no ordenadas. Las listas ordenadas están numeradas. Se representan con la etiqueta <ol> </ol> versión corta del inglés ordered list. Dentro de estas etiquetas se colocan otras etiquetas para enumerar la lista, estas son <li> </li>.

Ejemplo:

<ol> <li>Soy el item número uno</li> <li>Soy el ítem número dos</li> <li>Soy el ítem número tres</li> </ol>

Las listas desordenadas representadas por <ul> </ul>, donde ul es la versión corta del inglés "unordered list", crean listas que empiezan con puntos

Ejemplo

<ul> <li>Soy una lista no numerada</li> <li>No importa el orden</li> <li>No hay prioridades</li> </ul>

HTML también nos proporciona etiquetas para resaltar texto. Estas etiquetas son <strong></strong> para negritas y <em> </em> para itálicas.

Ejemplo:

Soy un párrafo, y me gustaría resaltar <strong>esta frase con negritas</strong>. También me gustaría enfatizar esta <em>palabra</em>con itálicas

Ya tenemos una idea de cómo estructurar una página web, con títulos, párrafos, imágenes, links y listas. Ahora es tiempo de agregarle un poco de estilo a la página, es decir, es tiempo de CSS.

Para empezar podemos cambiar el color del titulo principal. Para eso, se debe seleccionar el elemento que queremos cambiar, en este caso h1. Para cambiar estilos, CSS no requiere utilizar estos signos < > < / > . Sin embargo, CSS si requiere otros símbolos como los corchetes { }. El formato es el siguiente:

h1 {
color: red;
}

En este ejemplo, seleccionamos el elemento que queríamos cambiar, h1, y luego le agregamos el estilo, es decir el color, y le asignamos el color rojo. Nunca te olvides de agregar un punto y coma al final de cada estilo. Los estilos van siempre dentro de corchetes. De esta manera hicimos que el título principal sea en color rojo.

También podemos cambiar el color del fondo osea el "background" en inglés. Para eso, seleccionamos un elemento, por ejemplo la p, y le cambiamos el estilo dentro de corchetes:

p {
background-color: green;
}

Podemos cambiar varios estilos a la vez, colocando todos estos estilos dentro del mismo corchete. Por ejemplo, para cambiar el tamaño y tipo de letra del texto, podemos utilizar los estilos font-size y font-family dentro de un mismo corchete.

p {
font-size: 15px;
font-family: Open Sans;
}

También podemos agregar bordes a nuestros elementos en HTML. Por ejemplo, si queremos agregar un borde a la imágen, entonces seleccionamos img y agregamos los estilos que queramos dentro de los corchetes { }. Los bordes tienen tres atributos: color, ancho y estilo.

img {
border-color: red;
border-width: 5px;
border-style: solid;
}

La imágen tendrá un borde de color rojo, con un estilo sólido, y un ancho de 5px (pixeles).
