Hasta ahora vimos acerca de los elementos y estilos básicos para construir una página web. En esta lección, aprenderemos sobre las etiquetas divs y spans, las cuales nos ayudan a estructurar mejor una página web.

Las etiquetas <div> </div> nos permiten dividir la página en secciones. Funcionan como contenedores o bloques a los cuales se les puede agregar estilos diferentes. El mismo nombre de div es la versión corta de la palabra división.

Por ejemplo, si querés construir el sitio web de una tienda de vestimenta femenina, podes dividir la página en varias secciones gracias a <div></div>, una de ellas puede tener fotos de las prendas, otra sección puede tener alguna descripción de la tienda, y en la última sección se pueden encontrar los números de contacto telefónicos de la tienda. Cada sección puede tener colores y otros estilos diferentes que los identifiquen.

Por otro lado, las etiquetas <span> </span> te permiten cambiar el estilo de las partes más pequeñas del sitio web, como por ejemplo una palabra específica. Lo único que tienes que hacer es rodear el texto que necesitas cambiar con las etiquetas de span para luego agregar o modificar el estilo con CSS. Abajo tenemos un ejemplo:

<p>Estoy aprendiendo cómo <span>construir</span> páginas web</p>

span {
color: red;
}

Existen dos tipos de selectores que son muy útiles para agregar estilos en CSS: las clases (class) y los identificadores (id). Las clases sirven para seleccionar elementos que deberían tener el mismo estilo. Por ejemplo, imagina que tenemos tres secciones en una página web, y queremos que dos de ellas tengan el mismo color, y una de ellas tenga un color diferente.

Ejemplo:

<div> <h2>Esta sección tiene que ser verde</h2> </div> <div> <h2>Esta sección tiene que ser amarilla</h2> </div> <div> <h2>Esta sección tiene que ser verde</h2> </div>

Si utilizamos el elemento div para cambiar el estilo en CSS, todas las secciones representadas por div van a tener el mismo color. El cambio les afectará a todas las secciones por igual. Sin embargo, si discriminamos a las secciones, en este caso por color, agregando el selector class a las etiquetas div, entonces va a ser mucho más fácil hacer cambios en CSS.

Ejemplo:
<div class=”verde”>
<h2>Esta sección tiene que ser verde</h2>
</div>
<div class=”amarillo”>
<h2>Esta sección tiene que ser amarilla</h2>
</div>
<div class=”verde”>
<h2>Esta sección tiene que ser verde</h2>
</div>

class siempre va seguido del signo = y un nombre descriptivo entre comillas. La misma clase se puede utilizar varias veces; en este caso tenemos dos clases con el nombre “verde”. Ahora agregamos los estilos en CSS utilizando las clases. Para que CSS sepa que queremos agregar un estilo utilizando una clase, debemos poner un punto antes del nombre de la clase, y luego el procedimiento normal, es decir, corchetes con los estilos.

Ejemplo:
.verde {
background-color: green;
} .amarillo {
background-color: yellow;
}

Al igual que las clases, los identificadores sirven para agregar estilos a los elementos. Sin embargo, el mismo identificador se puede utilizar una sola vez en toda la página. Para asignar un identificador a un elemento HTML lo hacemos utilizando la palabra id seguida del signo = y un nombre entre comillas.

Ejemplo:

<div id=”row1”>
<h2>Fila 1</h2>
</div>
<div id=”row2”>
<h2>Fila 2</h2>
</div>

CSS reconoce a los identificadores con el signo #. Luego del signo numeral se coloca el nombre del identificador, seguido por los corchetes y respectivos estilos.

Ejemplo
#row1 {
font-size: 15px;
color: red;
font-style: Monospace;
}
#row2 {
font-size: 20px;
background-color: pink;
font-style: Open Sans;
}
